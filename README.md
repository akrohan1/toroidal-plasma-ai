## 🧠 Overview
This project explores plasma stability using toroidal and poloidal magnetic fields.
The goal is to improve confinement efficiency using AI-assisted optimization models.

- Toroidal + poloidal magnetic field structure
- Simulation-based approach
- AI-assisted control (concept phase)

## 🎯 Goal
Reduce plasma instability and energy loss in confined systems.
